# Smart Med V2

## Propósito
Aplicación móvil (y potencialmente web/desktop) desarrollada en Flutter para la administración práctica del botiquín del hogar. Permite gestionar medicamentos, mantener un inventario privado y buscar medicamentos por nombre o malestar.

## Estado Actual
**Phase 1 bootstrap** - Configuración inicial del proyecto.

## Nota Importante
**Smart Med V1 se conserva intacta** como referencia funcional y futura fuente de código reutilizable. Smart Med V2 es un proyecto completamente nuevo y se está construyendo desde cero absoluto.

## Stack Base
- **Framework:** Flutter
- **Lenguaje:** Dart
